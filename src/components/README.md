# Componentes

- Esta carpeta contiene todos los componentes reutilizables y específicos de la aplicación.
- Subcarpetas como `CoachDashboard/` y `ModelWorkoutModal/` agrupan componentes relacionados.
- Mantén los componentes pequeños y reutilizables.
- Si un componente crece demasiado, divídelo en subcomponentes.
